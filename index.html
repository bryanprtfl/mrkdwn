        POST
        /api/convert/url
      

      

        Converti un URL in Markdown.


        Body (JSON): { "url": "https://..." }

        Response: { "success": true, "markdown": "..." }
      

    


    

      

        POST
        /api/convert/file
      

      

        Converti un file in Markdown.


        Body: multipart/form-data — campo file

        Response: { "success": true, "filename": "...", "markdown": "..." }
